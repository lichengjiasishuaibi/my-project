# my-project：我的电子学习成长记录网站
## 🌟 项目简介
本项目是一个纯HTML+CSS实现的个人静态网站，用于记录我从零基础学习电子技术的完整成长历程，既是学习笔记的整理，也是个人作品的展示平台。网站采用了深色科技主题设计，通过时间线的形式，清晰呈现了我从入门到实践的每一个阶段。

## 📚 学习阶段记录
网站包含以下学习与实践阶段：
- ✅ 第一阶段：个人网站搭建，开启学习记录
- ✅ 第二阶段：初识3D打印，完成首个实体作品
- ✅ 第三阶段：电子焊接技术学习，掌握硬件基本功
- ✅ 第四阶段：嵌入式开发入门，制作WiFi控制南瓜灯
- ✅ 第五阶段：PCB设计进阶，完成首个自制电路板
- ✅ 第六阶段：激光切割技术入门，掌握EagleWorks软件
- ✅ 第七阶段：期末综合设计，实现双人下棋计时器

## 🛠️ 技术栈
- 前端：HTML5 + CSS3 + JavaScript
- 部署：支持Gitee Pages / Vercel / GitHub Pages等静态托管平台
